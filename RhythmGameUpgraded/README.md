# 2. Abgabe: PyBadge Rhythm Game

**Abgabe bis: 12. Mai**

## 1) Anpassung des Rhythm Games

- **Aufgabe:** Das Rhythm Game so anpassen, dass nach der Schwierigkeitsauswahl:
  - der Bildschirm schwarz wird, bevor das eigentliche Spiel startet
  - die **Neopixel-LEDs** anstelle des Displays verwendet werden, um Spielsignale anzuzeigen  
    - Falls die Neopixel nicht funktionieren, kann alternativ die **D13-LED** genutzt werden
- **Abgabe:**  
  - `.ino` Datei
  - kurzes Demo-Video (`.mp4`)

**Hinweis:**  
Die LEDs zeigen die aktuelle Spielaufgabe an (z.B. grün für einfache Beats, rot für schwierige Beats).
